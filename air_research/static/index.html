How AIR works

      Layer 1 — LLM emits compact JSON spec (not code).

      Layer 2 — Runtime reconstructs full HTML from templates.

      Layer 3 — Sandbox verifies output; repairs if needed.

      Result — verified output delivered, not just generated.
