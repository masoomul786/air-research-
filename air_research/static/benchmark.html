Syntactic Entropy Analysis — Htotal = Hsemantic + Hsyntactic

Boilerplate Ratio vs AIR Token Gain — by Domain

Entropy Decomposition — Research Framing
AIR's core insight: LLM output entropy has two separable components.
H_total = H_semantic + H_syntactic
H_semantic — user-specific content: data values, labels, copy text. Irreducible; must be generated.
H_syntactic — structural boilerplate: HTML scaffolding, Chart.js config objects, CSS layout, repeated patterns. These tokens are predictable and compressible.

AIR compresses the semantic layer into a compact JSON instruction, then reconstructs the syntactic layer via deterministic templates — entirely skipping LLM generation of predictable tokens.
